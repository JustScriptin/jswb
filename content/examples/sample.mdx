### Code Blocks

Here's a JavaScript example:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("World");
```

And a TypeScript example:

```typescript
interface User {
  name: string;
  age: number;
}

const user: User = {
  name: "John",
  age: 30,
};
```
