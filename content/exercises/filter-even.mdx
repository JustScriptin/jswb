# Filter Even Numbers

### Problem
Return only the even numbers from an array using `filter`.

### Example
```js
Input: [1, 2, 3, 4]
Output: [2, 4]
```

### Requirements
1. Use Array.prototype.filter()
2. Return an empty array if there are no even numbers
3. Handle negative numbers properly
